# Vibe Unity Developer

Готовый локальный ассистент для Unity лежит в `vibe-unity-developer`.

Быстрый запуск:

1. Открой `vibe-unity-developer/start-vibe-dev.cmd`.
2. В браузере открой `http://localhost:17861`.
3. Установи Unity-плагин через `vibe-unity-developer/install-plugin.cmd`.
4. В Unity открой `Window > Vibe Coding > Fullstack Developer`.
5. Нажми `Sync All Files`, включи `Poll Commands`, затем пиши задачи в браузере.

Подробно: `vibe-unity-developer/README_RU.md`.
