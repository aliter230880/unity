# AliTerra AI Coder

Unity Editor plugin for AliTerra metaverse — AI-powered scene editing and code generation.

## Plugin

Download `AliTerraAI.cs` and place it in `Assets/Editor/` in your Unity project.

- Menu: `Window → AliTerra → AI Coder` (Ctrl+Shift+A)
- Describe what to change in the scene — AI generates and applies the code automatically
